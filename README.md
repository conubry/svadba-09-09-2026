## Обновить изображение
1. Замени `invitation.png`
2. Пересобери оптимизации: `cwebp -q 85 invitation.png -o invitation.webp`
3. При смене композиции — пересобери `preview.jpg`
4. `git add -A && git commit -m "Update invitation" && git push`
